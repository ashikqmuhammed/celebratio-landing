# Celebratio Landing Page

Marketing landing page for Celebratio built with Next.js App Router.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Mobile-first responsive design

## Design System

Uses the same design tokens from the Celebratio dashboard:

- Brand colors (purple, pink, orange, blue)
- Spacing scale (xs to 3xl)
- Border radius (sm to 3xl)
- Shadows (sm to 2xl)
- Typography (Inter + Manrope)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
