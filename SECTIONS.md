# Landing Page Sections Overview

## 1. Navigation (Fixed Header)

- Sticky navigation bar with backdrop blur
- Desktop: Logo + Links + Sign In + CTA button
- Mobile: Hamburger menu with slide-down animation
- Smooth scroll to sections

## 2. Hero Section

- Full-screen hero with animated gradient background
- Floating gradient orbs (purple & pink)
- Animated badge with pulse effect
- Large headline with gradient text
- Subheadline explaining value proposition
- Two CTAs: "Start Free Trial" + "Watch Demo"
- Social proof: "Trusted by 10,000+ businesses"
- Placeholder logo grid
- Scroll indicator (desktop only)

## 3. Features Section

- Section header with gradient text
- Three-column grid (stacks on mobile)
- Each feature card:
  - Emoji icon with gradient background
  - Title and description
  - Hover effects (scale, shadow)
  - "Learn more" link with animated arrow
- Features: CRM, Website Builder, Marketplace
- Bottom CTA: "Explore All Features"

## 4. CRM Section

- Two-column layout (content + visual)
- Left: Content with badge, headline, description
- Feature list with checkmarks
- CTA button
- Right: Mockup of CRM dashboard
  - Pipeline visualization
  - Animated stats cards
  - Floating conversion rate badge

## 5. Website Builder Section

- Two-column layout (visual + content)
- Left: Browser window mockup
  - Browser chrome with traffic lights
  - Website preview with gradient hero
  - Content blocks
  - Floating "Live Editor" badge
  - "500+ Templates" badge
- Right: Content with features list

## 6. Marketplace Section

- Two-column layout (content + visual)
- Left: Content with stats grid
  - 10K+ Active Businesses
  - $2M+ Monthly Transactions
  - 50K+ Customer Reviews
- Benefits checklist
- Right: Product cards grid (2x2)
  - Each card: emoji, title, price, rating
  - Hover animations
  - Floating "1,000+ sellers online" badge

## 7. Pricing Section

- Section header
- Three pricing tiers (side-by-side on desktop)
- Middle tier highlighted as "Most Popular"
- Each card:
  - Plan name and description
  - Large price display
  - CTA button
  - Feature list with checkmarks
- Bottom note: "14-day free trial"

## 8. Testimonials Section

- Section header
- Three testimonial cards
- Each card:
  - 5-star rating
  - Customer quote
  - Avatar (emoji)
  - Name and role
- Hover effects

## 9. Final CTA Section

- Full-width gradient background (purple → pink → orange)
- Animated background orbs
- Large headline: "Ready to Grow Your Business?"
- Subheadline with social proof
- Two CTAs: "Start Free Trial" + "Schedule Demo"
- Trust badges:
  - 14-day free trial
  - No credit card required
  - Cancel anytime

## 10. Footer

- Dark background (gray-900)
- Five columns:
  - Brand + description + social links
  - Product links
  - Company links
  - Resources links
  - Legal links
- Bottom bar with copyright and legal links
- Social media icons

## Design Consistency

All sections follow:

- Mobile-first responsive design
- Consistent spacing (16px mobile, 24-32px desktop)
- Brand color gradients (purple, pink, orange)
- Smooth animations with Framer Motion
- Viewport-triggered animations
- Hover states on interactive elements
- Dark mode support throughout
